## 如何运行

```bash
export OPENSEARCH_HOST=<CDK 输出的 OpenSearch endpoint>
export AWS_DEFAULT_REGION=us-east-1

# 完整评测（生成 + 评分，约 5-8 分钟）
python3 eval/eval.py

# 仅重新评分
python3 eval/eval.py --score

# 拒答阈值标定
python3 eval/calibrate_tau.py
```

> `--score` 单独复评时 LLM judge 单次采样噪声较大（±2 分），不适合版本对比；应使用完整 `eval.py`。

---

## LLM-as-judge 评分规则

| 分段 | 含义 |
|------|------|
| 10 | 完全正确，引用来源准确，无遗漏 |
| 7-9 | 主要信息正确，少量遗漏或表述差异 |
| 4-6 | 部分正确，有明显遗漏或不准确 |
| 1-3 | 大部分错误或答非所问 |

特殊规则：库外题正确拒答 9 分；L3 题只返回单一来源最高 6 分。

## 调优方向

| 目标 | 调的参数 | 重点题型 |
|------|---------|---------|
| 提高 L1 命中率 | K_VECTOR / K_BM25 / TOP_N | q01~q05 |
| 提高 L2 综合质量 | TOP_M / chunk overlap | q06~q10 |
| 提高 L3 矛盾分析 | TAU_GAP / System Prompt | q11~q15 |
| 减少 OOS 漏拒 | TAU_ABS / BM25_FALLBACK_FLOOR | oos01~oos05 |
